Instalação do git
    sudo apt-get install git

Download do respositório
    git clone https://github.com/LionSec/katoolin.git

Envio da pasta para outro diretório e fazendo o executável
    sudo cp katoolin/katoolin.py /usr/bin/katoolin 
    cd /usr/bin/katoolin
    sudo chmod +x /usr/bin/katoolin  

Rodando o app
    sudo katoolin

Selecionar:
    Opção 1 para adicionar o repositório
    Update
    Opção 2 para escolher ferramentas
    Opção 0 para instalar todas

Tempo médio de instalaço: 1h30min
